# 消息定义

## HTTP 状态码

?> 本项目中的 HTTP 状态码仅下列值是受控的,其余状态码在本项目中均不具有业务意义。

| 值  | 描述     |
| --- | -------- |
| 200 | 成功     |
| 204 | 无内容   |
| 400 | 失败     |
| 401 | 未登录   |
| 403 | 未授权   |
| 500 | 内部错误 |

## 消息码

?> 1.消息码是可任意扩展的。  
2.消息码应该配合业务使用异常抛出。  
3.自定义业务异常应该继承`HandelException`类

消息码的定义位置：`QuickFrame.Common.MessageCodeOption`

| 值     | 描述                |
| ------ | ------------------- |
| 200    | 成功                |
| 400    | 失败                |
| 401    | 未登录              |
| 403    | 未授权              |
| 410    | 审核失败            |
| 411    | 弃审失败            |
| 412    | 审批失败            |
| 413    | 弃批失败            |
| 500    | 内部错误            |
| 550    | 任务超时            |
| 400001 | 请求参数错误        |
| 400002 | 查询条件语法错误    |
| 400003 | 数据已被修改,请重试 |
| 400004 | 数据不存在          |
| 400005 | 格式错误            |
| 400006 | Token 已过期        |
| 400007 | Token 无效          |
